Add real vine cutout assets here to replace the CSS-built fallback vines.

The component auto-detects these files and switches to them when present:

- `left-vine.webp` or `left-vine.png`
- `right-vine.webp` or `right-vine.png`
- `top-vine.webp` or `top-vine.png`
- `bottom-vine.webp` or `bottom-vine.png`

Use transparent-background cutouts sized for edge framing:

- Left/right: tall vertical crops
- Top/bottom: wide horizontal crops

WebP is preferred. PNG also works.
